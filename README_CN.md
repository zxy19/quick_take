# Quick Take

一个客户端 NeoForge 模组，为创造模式添加了 **Spotlight/Alfred 风格的快速物品搜索**界面。通过输入物品名称、注册 ID 或提示文本快速查找并选中物品。

![演示](demo.gif)

## 功能特性

- **即时物品搜索** — 在创造模式下按 `R` 键（可配置）打开全屏搜索界面
- **智能子序列匹配** — 输入 `dia pic` 即可找到"钻石镐"；搜索匹配显示名称、注册 ID 和提示文本
- **三阶段异步搜索** — 结果实时呈现：先显示名称匹配，再显示注册 ID 匹配，最后显示提示文本匹配
- **自动快捷栏放置** — 选中的物品会被智能放入快捷栏（复用已有物品、自动交换等）
- **输入快捷方式** — 在创造物品栏界面直接输入任意字符，即可自动打开搜索并预填该字符
- **拼音支持** — 可选集成 [Just Enough Characters](https://www.curseforge.com/minecraft/mc-mods/just-enough-characters)，支持中文拼音输入匹配

## 使用方法

1. 在创造模式下**按 R 键**打开搜索界面
2. 输入物品名称、注册 ID 或提示文本的部分内容 — 结果实时过滤
3. 使用方向键（↑↓）导航，按 Enter 选中，或直接用鼠标点击物品
4. 滚动鼠标滚轮查看更多结果
5. 按 Esc 关闭搜索界面

## 按键绑定

| 按键 | 默认键 | 说明 |
|------|--------|------|
| 打开 Quick Take | R | 打开搜索界面（仅创造模式） |
